# Java常用的对象工具类
常用的对象工具类

## 对象为空
```java
    /**
     * 判断字符串为空
     *
     * @param obj
     * @return
     */
    public boolean isNull(Object obj) {
        return Objects.isNull(obj);
    }
```

## 对象非空
```java
    /**
     * 判断字符串非空
     *
     * @param obj
     * @return
     */
    public boolean nonNull(Object obj) {
        return Objects.nonNull(obj);
    }

```

## 判断对象相等
```java
    /**
     * 判断字符串相等
     * @param a
     * @param b
     * @return
     */
    public boolean equals(Object a, Object b) {
        return Objects.equals(a, b);
    }
```
